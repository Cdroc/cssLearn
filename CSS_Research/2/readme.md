1.基本选择器
    1.1标记选择器 -- h1     -- 适用于"同组"标记的样式设置
    1.2类别选择器 -- .class -- 适用于"所有"标记的样式设置
    1.3ID选择器   -- #id    -- 适用于"单一"标记的样式设置
2.复合选择器
    2.1交集选择器 -- h1.class/h1.id
                  -- 第一个必须是标记选择器，第二个必须是类别选择器或ID选择器
    2.2并集选择器 -- h1, .class, #id, h1.class, h1.id, p span b
                  -- 任何选择器都可以作为并集选择器的一部分
    2.3后代选择器 -- p span b
                  -- <p>里面嵌套了<span>，<span>里面嵌套了<b>
3.通用选择器      -- *
4.样式优先级      -- 行内样式 > ID样式 > 类别样式 > 标记样式
