1.行内式
    行内式即在标记的style属性中设定CSS样式，这种方式本质上没有体现CSS的优势，因此不推荐使用。
2.嵌入式
    嵌入式则将对页面中各种元素的设置集中写在<head>和</head>之间，适用于单一页面，对于多页面网站就失去了CSS的巨大优点。
3.导入式
    导入式使用CSS的规则引入外部CSS file，是在整个页面装载完成后再装载CSS file，如果网页件体积较大，则会出现先显示没有样式的页面，闪烁一下后再出现设置样式后的效果。
4.链接式
    链接式使用HTML的标记引入外部CSS file，是在装载页面主体部分之前装载CSS file,这样显示出来的网页从一开始就是带有样式效果的。
